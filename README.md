Student Result Management System

Project Overview
This project is built using MySQL to manage student records, marks, subjects, and attendance.

Features
- Student management
- Marks management
- Attendance tracking
- Grade calculation
- Performance analysis

Technologies Used
- MySQL
- SQL
- phpMyAdmin
- XAMPP

SQL Concepts Used
- Joins
- Aggregate Functions
- Group By
- CRUD Operations
- CASE Statements

Sample Queries
- Find topper
- Calculate average marks
- Attendance report
- Grade generation
